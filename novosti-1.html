VAŽNOST SSL CERTIFIKATA ZA VAŠU WEB STRANICU ILI WEB SHOP

Kao vlasnik web stranice, zasigurno ste se pitali što možete učiniti kako bi vaša web stranica bila sigurna za vas i vaše klijente. U ovom članku možete pročitati kako pretvoriti svoju web stranicu u sigurno mjesto za posjetitelje i kupce.

Za početak, znate li što predstavlja lokot kraj URL-a vaše web stranice?

To je kriptografski protokol Secure Sockets Layer (SSL). Pojednostavljeno, SSL pruža siguran kanal na internetu između dva ili više uređaja. Ili u/na internoj mreži.

Najreprezentativnija je veza između internetskog preglednika i internet poslužitelja. Na primjer, između vašeg internet preglednika na osobnom ili prijenosnom računalu i poslužitelja vaše banke.

Ako je web stranica osigurana SSL-om njegova se adresa mijenja iz HTTP-a u HTTPS. Sufiks “S” znači “sigurno”.

SSL certifikat

SSL certifikat sastoji se od malih podatkovnih datoteka koje digitalno povezuju kriptografski ključ s pojedinostima određene organizacije.

Kada je instaliran na web poslužitelju, omogućuje prikaz lokotića/lokota i https protokol koji dopušta sigurnu vezu s web poslužitelja na preglednik.

SSL se obično koristi za sigurne transakcije kreditnim karticama, prijenosa podataka i prijava u račune, a postaje i uobičajen prilikom pretraživanja društvenih medija.

Prednosti SSL certifikata

Od 68. Verzije preglednika, objavljene u srpnju 2018. godine, Chrome označuje sve adrese do kojih se ne može doći putem enkriptirane veze, kao nesigurne. To će također utjecati na vašu poziciju na Googleu. Uz sve to krađa podataka postaje gotovo nemoguća.

Bez šifriranja, online prevaranti/kriminalci bi lakše mogli presretati podatke poput osobnih adresa ili brojeva kreditnih kartica i tako ih zloupotrebljavati. Cilj je da se cijeli web pretvori u sigurno mjesto.

To je također način na koji Google svojim korisnicima omogućuje prestanak posjeta nesigurnih web stranica te poticaj programerima da krenu izrađivati samo sigurne web stranice.

Od srpnja 2018. sve ne-šifrirane web stranice označene su kao nesigurne.

To je loše za vas kao poduzetnike i vlasnike web stranica. Posjetitelji se neće dugo zadržavati na takvim web stranicama i podijeliti svoje informacije. Pogotovo ako se radi o web shopu.

SSL certifikat može utjecati i na brzinu učitavanja vaše web stranice. Od 2015. upotrebljava se novi HTTP / 2, a podržan je samo u web preglednicima ako je veza kriptirana SSL-om. Sve web stranice koje imaju HTTP / 2 otvaraju se u pregledniku puno brže.

Važno imati na umu da od svibnja 2018., prema GDPR-u, vaša je zakonska obveza imati sigurne web stranice za svoje posjetitelje i kupce.

Izrada SSL certifikata

Prvi korak za upotrebu novog weba trebala bi biti kupnja SSL certifikata. Važno za znati: SSL certifikati vrijede samo određeno vremensko razdoblje i moraju se obnoviti nakon isteka roka trajanja.

SSL certifikati dostupni su u tri varijante:

Domain Validation Certificate (DV): DVC potvrđuje da je promet na stranici šifriran/enkriptiran i da se prikazana domena podudara s onom traženom. DV certifikati dostupni su gotovo odmah.

Organization Validation Certificate (OV): prosječna razina enkripcije: CA (Authority Certificate) provjerava pripada li domena zapravo (vašoj) tvrtki, a ne nekom drugom vlasniku. Uz OV certifikat, kupci su osigurani da su uz željenu, legitimnu tvrtku.

Extended Validation Certificate (EV) – Najviša razina enkripcije: za EVC, vlasnik domene mora proći opsežnije provjere. Uz lokot, naziv tvrtke prikazan je zelenom bojom uz EV certifikat na svim uobičajenim preglednicima. EV certifikati najbolje odgovaraju kada su u pitanju prijenosi osjetljivih podataka kao što su bankovni podaci ili medicinske informacije.

OpenSSL

OpenSSL je alat za programe koji štite online komunikaciju od monitoringa ili bilo kojih ostalih ilegalnih radnji.

OpenSSL je komercijalni alat za Transport Layer Security (TLS) i Secure Sockets Layer (SSL) protokole. Također je označen kao universal cryptography library.

Ako radite s TLS i SSL protokolima, radit ćete s OpenSSL-om. Ovaj odličan alat pomaže vam u izradi SSL certifikata.

Međutim, prvo morate imati pristup stranici Cloud Profile Management na Cloud Management Console-i.

Zaključak

Svaka moderna web stranica mora imati SSL certifikat. Pomaže vam u zaštititi vaše web stranice te da privučete više posjetitelja.

Također je važno za web shopove da se podaci vaših klijenata sigurnosno pohranjuju kako bi kupci imali dovoljno povjerenja za kupovinu kod vas.

Ako imate pitanja o SSL-u i TLS-u, kontaktirajte nas u bilo kojem trenutku.
